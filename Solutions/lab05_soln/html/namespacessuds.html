Detailed Description

S.S.U.D.S. (SSU Data Structures) is our data structures implementation of common data structures. In general, these classes are probably less good than the STL equivalents (vectors > ArrayList, linked_list > LinkedList, etc.). There are a few data structures that don't (currently) have a STL equivalent.
