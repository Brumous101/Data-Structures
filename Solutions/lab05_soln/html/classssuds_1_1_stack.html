Detailed Description

template<class T>
class ssuds::Stack< T >

A Stack is a simple LIFO data structure (last-in-first-out). Here I implement it using linked list as these support fast push / pop from the ends of the list, which is what you'll primarily use with a stack. For illustration, I implemented this class using a "has-a" (rather than a "is-a" / inheritance) approach.
